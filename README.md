# SimulAnts 🐜

SimulAnts is an ant colony simulation project written in Java, with a planned browser-based visual demo using JavaScript and GitHub Pages.

The goal is to simulate ants exploring a world, finding food, following pheromone trails, and building colony behavior over time.

## Features

- Tile-based world grid
- Queen ant spawning logic
- Worker ants with task-based behavior
- Food and home pheromone system
- Pheromone decay and diffusion
- Colony food storage
- Experimental AI-like ant decision logic

## Planned Features

- Soldier ants
- Better pathfinding
- More advanced pheromone behavior
- Colony expansion
- Visual JavaScript demo
- Browser version at `simulants.kartibrown.com`
